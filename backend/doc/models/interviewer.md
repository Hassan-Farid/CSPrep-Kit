
# Interviewer

Provides random question for preparation for CS Interviews

## Structure

`Interviewer`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `name` | `string` | Required | Question |
| `link` | `string` | Required | Link to explore answer to question |
